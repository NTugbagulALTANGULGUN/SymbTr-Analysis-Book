# Primary Dataset

## SymbTr v3.0 Turkish Maqam Music Dataset

**DOI**

https://doi.org/10.5281/zenodo.15470412

The analyses presented throughout this book are primarily based on the **SymbTr v3.0** corpus, one of the largest publicly available symbolic Turkish makam music datasets.

Dataset statistics presented throughout this book are generated **dynamically** from the dataset and therefore automatically reflect future dataset updates.

### Dataset Used in This Book

The computational analyses presented throughout this book use the following file:

**Turkish maqam music pieces in time series format_3000 rows x 365 columns.csv**

This dataset contains pitch trajectory representations of Turkish makam music pieces and serves as the primary input for all statistical analyses, visualizations, dimensionality reduction, and clustering experiments.

# Related Turkish Makam Music Datasets

## Turkish Makam Symbolic Phrase Dataset

A machine-readable collection of Turkish makam music scores segmented into symbolic phrases by expert annotators.

Project page:

http://compmusic.upf.edu/node/237

---

## Turkish Makam Melodic Phrase Dataset

A phrase-level symbolic dataset manually annotated by domain experts for melodic segmentation and computational analysis.

Project page:

http://compmusic.upf.edu/node/236
