# Introduction

Turkish maqam music is a rich modal music tradition characterized by distinctive melodic structures, pitch organizations, and performance practices.

This book explores the SymbTr v3.0 Turkish Maqam Music Dataset through data science and computational musicology methods.

The dataset contains 3000 musical pieces represented as time-series pitch trajectories with 365 normalized time steps.

The analysis presented in this book includes:

- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualization
- Principal Component Analysis (PCA)
- K-Means Clustering
- Cluster Interpretation

The goal is to explore patterns, similarities, and structures that emerge from the data.
