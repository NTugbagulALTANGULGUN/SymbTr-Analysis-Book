The corpus is intended to support research in music information retrieval (MIR), computational musicology, and machine learning applied to Turkish classical and folk music.

## What is this book about?

This book project "Turkish Delight Corpus (TDC): A Repository for Turkish Makam Music Research and Computational Methods” focuses on statistical analysis, visualization, dimensionality reduction, clustering, and cluster interpretation of Turkish maqam music pieces represented as time-series pitch trajectories.

## Project Workflow

Topics covered include:
* Dataset acquisition and validation
* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Data Visualization
* Principal Component Analysis (PCA)
* K-Means Clustering
* Cluster Interpretation
