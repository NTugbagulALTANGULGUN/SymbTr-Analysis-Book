# Welcome!

This online book presents a computational exploration of the SymbTr v3.0 Turkish Maqam Music Dataset.

The project focuses on statistical analysis, visualization, dimensionality reduction, clustering, and cluster interpretation of Turkish maqam music pieces represented as time-series pitch trajectories.

## What is this book about?

This book demonstrates how data science techniques can be applied to Turkish maqam music.

## Project Workflow

Topics covered include:
* Dataset acquisition and validation
* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Data Visualization
* Principal Component Analysis (PCA)
* K-Means Clustering
* Cluster Interpretation

## Dataset

SymbTr v3.0 Turkish Maqam Music Dataset

https://zenodo.org/records/15470412
